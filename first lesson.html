Let's talk about what it is, this Arduino ü§î. Arduino is a simple electronic tool that makes electronic work easy. Arduino is a development board brand. There are hundreds of other types of development boards in the world. Arduino, Gavesha, Magicbit, Micro: bit, NodeMCU, TTGO, Raspberry Pi, Orange Pi, etc. Among these, some are famous development boards. If you work with these development boards, we want to program them. So, to program the Arduino boards, we need to know the programming language specific to the Arduino platform. Arduino boards have a separate language made from C/C++ languages. Many people refer to this language as the Arduino language. And some people call this language the C/C++ language. We can use separate software to program the Arduino boards in this Arduino language. It is called the Arduino Software. But, we can also use Vs. Code (Visual Studio) for programming Arduino boards. I will give a lesson about it in the next class.

We will also talk about the advantages and disadvantages of Arduino boards.

Advantages:
1. You can use small programs to do your electronics work.
2. You can do electronics work without using ICs, transistors, and capacitors as components.
3. It is smaller in size.
4. Even children can learn it quickly and easily.
5. Wifi, Bluetooth, and protocols are easy to use with projects.

Disadvantages:
1. You need knowledge of programming.
2. Arduino boards are expensive these days.
3. You have to buy an Arduino board at a high cost to do a small electronics project.
4. Arduino platforms are not suitable for industrial-level projects.
5. If your Arduino board dead, you will have to buy a new one or replace the Atmega328p IC of your Arduino board.

There are many models of Arduino development boards üò≤. Arduino Uno, Arduino Nano, Arduino Micro, Arduino Pro mini, Arduino Pro micro, Arduino Leanado, Arduino Nano 101, Portenta H7 X8, Arduino Mega, etc.

The main important points of the Arduino Uno board's datasheet can be seen below üëá.

Microcontroller = Atmega328p IC
Operating Voltage = 5V
Input Voltage (Recommended) = 7V - 12V
Input Voltage (Limit) = 6V - 20V
Digital I/O Pins = 14 (of which 6 provide PWM outputs)
PWM Digital I/O Pins = 6
Analog Input Pins = 6
DC Current per I/O Pins = 20mA
DC Current from 3.3V Pins = 50mA
Flash Memory = 32KB (Atmega328p); the bootloader uses 0.5KB
SRAM = 2KB (Atmega328p)
EEPROM = 1KB (Atmega328p)
Clock Speed = 16MHz
LED Built-in = 13
Length = 68.6mm
Width = 53.4mm
Weight = 28g
ICSP Pins = MOSI, MISO, SCK, Reset, VCC, GND, SPI

Here, I/O Pins are Input Output Pins, ICSP is In-Circuit Serial Programming, SPI is Serial Peripheral Interface, SCK is Serial Clock, and it has an SDA pin. It is used for communication protocol that uses two wires, and you can connect up to 150 i2c sensors with those two wires üòÆ.

This datasheet's details are valid only for the original Arduino board.
